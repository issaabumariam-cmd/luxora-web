Executive Summary

A cinematic, responsive, 10-section landing page was designed and built for the LUXORA premium oral-care brand. The page uses a luxury glass/chrome aesthetic inspired by the provided reference images, with real product photos extracted via an AI background-removal pipeline, scroll-driven animations via Framer Motion, smooth scrolling via Lenis, and is deployed on Vercel’s free Hobby plan.

Known Constraints & Gotchas

  • Reference image filenames are misleading. hero-logo-reference.png actually contains the care kit tray; dental-care-kit-reference.png contains the chrome tooth hero. Always verify image content by opening the file.
  • Static export needs images.unoptimized: true. Next.js 16 still requires this for fully static export.
  • Do not use distDir: "dist". Next.js 16 dev server stores its cache under dist/dev/; the current config uses dist-export/ to avoid collision.
  • rembg BiRefNet model is ~973 MB. First extraction run downloads the model. Harmless Cholesky warnings may appear during alpha matting.
  • Some extracted PNGs may need normalization. If images fail to render in the browser, run python scripts/fix_pngs.py to re-save them cleanly with Pillow.
  • Keep cosmetic claims safe. Do not make unsubstantiated dental/medical claims. A disclaimer is included in the footer.
